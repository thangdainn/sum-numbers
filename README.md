# Cộng hai số lớn

## Giới thiệu
Thực hiện phép cộng hai số được biểu diễn dưới dạng chuỗi ký tự.  
Bài toán được triển khai giống cách học sinh Tiểu học cộng số thủ công: duyệt từ phải sang trái, ghi nhớ (carry) và ghi log chi tiết từng bước phép toán.

## Công nghệ sử dụng
- Java 21
- Maven 3
- JUnit 5 (Unit Testing)
- java.util.logging


## Cách sử dụng

### 1. Clone repository về máy
```bash
git clone https://github.com/thangdainn/sum-numbers.git
cd sum-numbers
````
### 2. Build project
```bash
mvn clean package
````
### 3. Chạy Unit Test
```bash
mvn test
